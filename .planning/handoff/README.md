## Conventions

- One handoff file per workstream (not per chat turn).
- Update the **Updated** column when you materially change a handoff.
- Archive or delete a handoff when the workstream is merged, shipped, or abandoned.
- Keep handoffs factual: paths, decisions, test status, next steps — not narrative.
