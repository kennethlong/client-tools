PRIORITY ORDER [TreeFile.cpp:285]:
  bool searchNodePriorityOrder(a, b) { return a->getPriority() > b->getPriority(); }
  → HIGHER priority NUMBER is searched FIRST. Stable sort preserves cfg insertion order

    sorted(nodes, key=lambda n: -n.priority)  (Python sort is stable.)

FIRST-MATCH-WINS [TreeFile.cpp:456]:
  iterate nodes in priority order; the FIRST node that has the path wins; later copies
  are "shadowed". Loop guard is `!deleted` — a tombstone hit stops the whole search.

fixUpFileName EXACT TRANSFORM [TreeFile.cpp:511-601], in order:
  1. strip ALL leading '\' or '/'
  2. strip leading './' or '.\' (repeatedly)
  3. strip leading '../' or '..\' (repeatedly)
  4. for the remainder: every '\' -> '/'; tolower() every non-slash char
  5. collapse repeated consecutive slashes to a single '/'

TOMBSTONE — PER-NODE-TYPE (the #1 nuance CONTEXT under-specified):
  SearchTree (.tre) length==0  [TreeFile_SearchNode.cpp:397]:
    sets deleted=true, returns "not found" → with find()'s !deleted guard the ENTIRE
    search STOPS; the file is removed globally. (override-delete mechanism)
  SearchTOC (.toc) length==0 OR offset==0  [TreeFile_SearchNode.cpp:807, :831]:
    returns "not here" but ALWAYS leaves deleted=false → "absent here, keep searching";
    does NOT shadow a real lower-priority copy.
  SearchPath (dir): no tombstone concept — filesystem existence only.

KEY-NAME GRAMMAR [TreeFile.cpp:113-115]:
  sprintf(skuText, "_%02d_", sku); sprintf(buffer, "searchTOC%s%d", skuText, priority);

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| binary archive entries → virtual-tree keys | Embedded filenames inside `.tre`/`.toc` are attacker-influenceable and become merge keys; a hostile `..`/absolute path could escape the virtual root |
| searchPath loose dir → os.walk | The override dir is walked; symlinks or paths could escape the declared dir |
| cfg text → scanner | `client.cfg` `[SharedFile]` values (paths, priorities) are read but not executed; malformed lines must not crash the parser |
