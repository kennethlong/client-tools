**State-cache architecture** (per PATTERNS §`Direct3d11_StateCache.{h,cpp}` lines 241-315):

D3D11 replaces D3D9's per-state-key setter pattern with immutable state objects:
- `ID3D11RasterizerState` — fill mode, cull mode, depth bias (from D3DRS_FILLMODE / D3DRS_CULLMODE / D3DRS_DEPTHBIAS)
- `ID3D11BlendState` — blend factors per render target (from D3DRS_SRCBLEND / D3DRS_DESTBLEND / D3DRS_BLENDOP / D3DRS_SRGBWRITEENABLE)
- `ID3D11DepthStencilState` — depth test, stencil ops (from D3DRS_ZENABLE / D3DRS_ZFUNC / D3DRS_STENCIL*)
- `ID3D11SamplerState` — texture filter, address mode, MIP LOD bias (from D3DSAMP_ADDRESSU/V / MAGFILTER / MINFILTER / MIPFILTER / MAXANISOTROPY / SRGBTEXTURE)

Cache each state object by hash of its descriptor; bind via `RSSetState` / `OMSetBlendState` / `OMSetDepthStencilState` / `PSSetSamplers` only when state changes.

**Pitfall 4 — SRV/sampler split** (RESEARCH lines 440-445 + PATTERNS lines 257-269):
D3D9's `SetTexture(stage, tex)` set both the resource and sampler in one call. D3D11 splits them:
- `PSSetShaderResources(slot, 1, &srv)` for the texture/resource view
- `PSSetSamplers(slot, 1, &sampler)` for the sampler state

The state cache must track BOTH bindings independently. Engine's `Gl_api::setGlobalTexture(stage, tex)` slot binds the SRV; the implicit sampler defaults come from the shader-implementation pass data (the `D3DSAMP_*` settings the D3D9 code reads from `.iff` shader templates).

**Draw-call mapping**:

| D3D9 method | D3D11 equivalent | Notes |
|-------------|------------------|-------|
| `DrawPrimitive(POINTLIST, ...)` | `Draw(vertexCount, startVertex)` after `IASetPrimitiveTopology(POINTLIST)` | |
| `DrawPrimitive(LINELIST, ...)` | `Draw(...)` after `IASetPrimitiveTopology(LINELIST)` | |
| `DrawPrimitive(TRIANGLELIST, n)` | `Draw(n*3, startVertex)` after `IASetPrimitiveTopology(TRIANGLELIST)` | |
| `DrawPrimitive(TRIANGLESTRIP, n)` | `Draw(n+2, startVertex)` after `IASetPrimitiveTopology(TRIANGLESTRIP)` | |
| `DrawIndexedPrimitive(TRIANGLELIST, ...)` | `DrawIndexed(indexCount, startIndex, baseVertex)` after `IASetPrimitiveTopology(TRIANGLELIST)` | |

Pre-draw setup (every draw call must ensure):
1. `IASetVertexBuffers(slot, count, &vb, &stride, &offset)` — bind VB
2. `IASetIndexBuffer(ib, format, offset)` — bind IB (R16_UINT or R32_UINT per per-buffer m_indexFormat from Plan 04)
3. `IASetInputLayout(layout)` — bind input layout (cached in VertexDeclarationMap)
4. `IASetPrimitiveTopology(...)` — point list / line list / triangle list / triangle strip
5. `VSSetShader(vs)` / `PSSetShader(ps)` — bind shaders
6. `VSSetConstantBuffers` / `PSSetConstantBuffers` — bind cbuffers
7. `PSSetShaderResources(slot, n, &srv)` + `PSSetSamplers(slot, n, &sampler)` — bind textures + samplers (Pitfall 4 split)
8. `RSSetState` / `OMSetBlendState` / `OMSetDepthStencilState` — bind state objects
9. `RSSetViewports(1, &vp)` — viewport (set once in beginScene; re-set on viewport change)
10. `Draw` or `DrawIndexed`

```cpp
namespace Direct3d11_VertexDeclarationMapNamespace
{
    struct Key {
        uint32_t vbFormatHash;
        uint8_t  vsBytecodeSha1[20];
        bool operator<(Key const &rhs) const {
            return memcmp(this, &rhs, sizeof(Key)) < 0;
        }
    };
    std::map<Key, Microsoft::WRL::ComPtr<ID3D11InputLayout>> ms_cache;
}

ID3D11InputLayout * Direct3d11_VertexDeclarationMap::getOrCreate(
    VertexBufferFormat const &fmt, ID3DBlob const *vsBytecode)
{
    Key key = {};
    key.vbFormatHash = /* hash fmt component flags */;
    // SHA1 the vsBytecode contents into key.vsBytecodeSha1
    computeSha1(vsBytecode->GetBufferPointer(), vsBytecode->GetBufferSize(), key.vsBytecodeSha1);
    
    auto it = ms_cache.find(key);
    if (it != ms_cache.end()) {
        Direct3d11_Metrics::inputLayoutCacheHits++;
        return it->second.Get();
    }
    Direct3d11_Metrics::inputLayoutCacheMisses++;
    
    D3D11_INPUT_ELEMENT_DESC desc[16];
    int count = Direct3d11_VertexBufferDescriptorMap::buildInputElementDesc(fmt, desc);
    
    Microsoft::WRL::ComPtr<ID3D11InputLayout> layout;
    HRESULT hr = Direct3d11_Device::getDevice()->CreateInputLayout(
        desc, count,
        vsBytecode->GetBufferPointer(), vsBytecode->GetBufferSize(),
        &layout);
    FATAL_DX_HR("CreateInputLayout failed", hr);
    
    ms_cache[key] = layout;
    return layout.Get();
}
```

{
    auto ctx = Direct3d11_Device::getContext();
    
    // 1. Input layout (may change on shader/format change)
    if (ms_currentInputLayout) ctx->IASetInputLayout(ms_currentInputLayout);
    
    // 2. Topology
    ctx->IASetPrimitiveTopology(ms_currentTopology);
    
    // 3. VB
    if (ms_currentVB) {
        ID3D11Buffer * vbs[1] = { ms_currentVB };
        UINT strides[1] = { ms_currentVBStride };
        UINT offsets[1] = { ms_currentVBOffset };
        ctx->IASetVertexBuffers(0, 1, vbs, strides, offsets);
    }
    
    // 4. IB
    if (ms_currentIB) {
