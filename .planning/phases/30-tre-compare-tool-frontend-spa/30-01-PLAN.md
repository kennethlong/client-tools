<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → FastAPI (127.0.0.1) | localhost-only; the single 127.0.0.1 bind is the access control (T-29-03-03) |
| FastAPI static mount → filesystem | StaticFiles serves files under web/build/ only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-30-01-01 | Information disclosure | SpaStaticFiles static mount | mitigate | `StaticFiles(directory=web/build)` confines serving to that dir; the 404→index.html fallback only ever re-serves `index.html` from that same dir — never join request paths to the filesystem manually. Starlette handles range/etag/mime safely. |
| T-30-01-02 | Elevation / Exposure | new listener / bind widening | mitigate | The SPA serve is additive INSIDE `create_app` (same app, same port) — no second listener; `__main__.py` HOST stays `127.0.0.1` (asserted in Task 3 acceptance). |
| T-30-01-03 | Tampering (route shadowing) | greedy `/` mount | mitigate | Mount registered LAST, after all four API routes; the mount-order regression test asserts `/compare/files` returns JSON not the SPA HTML. |
| T-30-01-04 | Tampering | npm supply chain | accept | Mainstream deps (Vite/React/TanStack/shadcn) pinned via `package-lock.json`; localhost dev tool, low-value target. |
</threat_model>
