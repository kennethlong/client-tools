THE LANDMINE RULES (unchanged from 37-02, applied across the whole full set):
- VIRTUAL → SKIP + `// SKIP: virtual — Utinni resolves off vtable (spec §6)`. Never raw &fn.
- CONSTRUCTOR → free-function thunk; never &Class::Class.

- INLINE → prefer non-inline; force emission only if required; else SKIP+document (no stable/unique address).
- UNCONFIRMABLE → OMIT + document (graceful degradation, EPA-04). Never guess.
