// Structure definition:
struct EAX_PSHIFTER
{
    S32 Slot;
    S32 Volume;
    S32 CoarseTune;
    S32 FineTune;
};
Description

used to control the pitch shifter filter in the EAX 4 3D provider

Members
Slot [in]

specifies the slot to query or set. The pitch shifter can use slot 2 and/or 3.
Volume [in]

specifies the volume of this slot in standard EAX attenuation units (-10000 to 0 fixed point millibels, default 0.0).
CoarseTune [in]

specifies the amount of pitch shift in semitones (-12 to 12, default 12).
FineTune [in]

specifies the amount of pitch shift in cents (-50 to 50, default 0).
Discussion

See the EAX 4 documentation for technical details on this filter.
