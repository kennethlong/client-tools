// Type Definition:
typedef U32 MILES_OPEN_DIGITAL_FLAGS;
Description

specifies open flags for the AIL_open_digital_driver function.

Predefined values
0 

Use the default settings. Under Windows, Miles will try DirectSound first, and then try waveOut if DirectSound fails (or is using emulation mode).

AIL_OPEN_DIGITAL_FORCE_PREFERENCE 

Under Windows, this flag tells AIL_open_digital_driver to obey the DIG_USE_WAVEOUT preference instead of trying DirectSound first and then waveOut automatically.
AIL_OPEN_DIGITAL_USE_SPU0 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 0. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU1 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 1. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU2 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 2. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU3 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 3. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU4 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 4. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU5 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 5. This should match the number that you passed to MilesStartAsyncThread.

AIL_OPEN_DIGITAL_USE_SPU6 

Under PS3, this flag tells AIL_open_digital_driver to process the digital driver on SPU index 6. This should match the number that you passed to MilesStartAsyncThread.
