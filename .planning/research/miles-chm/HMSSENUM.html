// Type Definition:
typedef SINTa HMSSENUM;
Description

specifies a type used to enumerate through a list of properties.

Predefined values
MSS_FIRST 

use this value to start the enumeration process. [Value: ((HMSSENUM)-1)]
Discussion

The Miles enumeration functions all work similarly - you set a local variable of type HMSSENUM to MSS_FIRST and then call the enumeration function until it returns 0.
