// Function prototype:

HTIMER AIL_register_timer(AILTIMERCB callback_fn);
Description

registers a callback function callback_fn to receive asynchronous time-slice service from a timer, and returns a handle HTIMER which may be used to start, stop, and program the function's assigned timer.

Parameters
callback_fn [in]

points to a timer callback function declared as the AILTIMERCB type.
Return value
return [out]

A timer handle which may be used to start, stop, and control the timer.

Discussion

A timer is an API resource which, can provide periodic callback service to a given function at any desired frequency within the limits of system overhead. This callback is made from a background thread.

AIL_register_timer will return the value -1 if no timer handles are available.

Timer callback service will not begin until the timer's service rate is established with AIL_set_timer_period or AIL_set_timer_frequency, and the AIL_start_timer function is called to initiate timer service.

The timer callback function's user argument receives the value passed by the application to the AIL_set_timer_user function.

The user argument allows a single function to be registered multiple times as a timer client, and called to handle timer processing for several different object instances. For example, an application may cast a structure pointer to type U32 and, by passing it to AIL_set_timer_user, cause the structure pointer to be passed on to that timer handle's timer service routine, which would then operate only on the structure whose pointer is passed to it by the MSS timer dispatcher. This is illustrative of the techniques MSS itself uses to manage multiple XMIDI or digital audio drivers simultaneously, and any application with an "object-oriented" design may benefit as well from the user feature.

Most MSS functions can be safely invoked from within timer callback functions. Exceptions are noted in the descriptions of the functions involved. If in doubt about whether a particular action may safely be performed from within a timer callback function, contact RAD Game Tools for assistance.
