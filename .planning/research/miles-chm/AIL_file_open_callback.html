// Function prototype:

typedef U32 AILCALLBACK AIL_file_open_callback(char const * filename,
UINTa* file_handle);
Description

specifies a user-replaceable callback handler to open a file in MSS.

Parameters
filename [in]

specifies the filename to open.
file_handle [out]

returns the new file handle, if the open succeeds.
Return value
return [out]

Returns non-zero, if the open succeeded. Zero, otherwise.

Discussion

Use these function definitions for your own file IO callbacks. Use AIL_set_file_callbacks to install them.
