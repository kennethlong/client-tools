Miles Studio 9.2 is an incremental upgrade with several new features.

  • Busses - You can now define a label query that will cause all sounds to mix to a separate buffer prior to final mix. The signal strength of each bus is routed to a variable you can use as though your game set it. See Busses.

  • Defaults - Avoid the hassle of creating a child asset to control compress on varying platforms! By setting your sound assets to Use Defaults, they will use platform specific settings. See Defaults.

  • Variable Control - Variables can be set to LFO in Miles Studio. In addition, a default value can be set, and the Move Variable event step can be used to vary the value over time. See Variables.

  • Selected Sounds ("Dialog Events") - By marking a sound as "Selected", the chosen sound file in the sound list will use an index supplied by the game code. See Start Sound: Sound Selection.
