Q: How to I set the compression format to use for a sound asset in my SoundBank? How about per platform?

A: There are several ways to set the compression format of a sound asset. The first, somewhat inflexible, way is that if you add a sound asset that is already compressed, then it will stay in that form. That is, only uncompressed sound assets are processed by the Miles Sound Studio - compressed assets are just compiled in directly.

Obviously that inflexible option is only for unusual situations (like an MP3 file for background music, say) - for most sounds, you want the Miles Sound Studio to process your asset as it is added to the soundbank and deployed for other platforms. In this case, just select the desired compression in the Sound Asset. Relevant options will appear, which you can adjust as needed.

To override this per platform, open the Asset Specialization for the Sound Asset. Click the button to add a new child asset (see the Asset Reference), and then select the child asset in the grid for the platforms you want to use it with. Then select the child asset in the list, and change the compression.
