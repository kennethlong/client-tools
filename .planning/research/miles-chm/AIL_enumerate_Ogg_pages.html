// Function prototype:

S32 AIL_enumerate_Ogg_pages(OGG_INFO* inspection_state);
Description

obtains information about successive pages in an Ogg container file.

Parameters
inspection_state [out]

points to an instance of an OGG_INFO structure owned by the application and previously initialized by a call to AIL_inspect_Ogg.
Return value
return [out]

1 if the page-data portion of the OGG_INFO structure represents a valid page; 0 if no (further) valid pages exist in the file image.

Discussion

This function returns information about all of the valid Ogg pages in the file image, beginning with the first, by filling in the appropriate OGG_INFO structure fields.

You can inspect all of the pages in an Ogg file image with the following code: 

   OGG_INFO info;
   AIL_inspect_Ogg(&info,
                   my_Ogg_file_image,
                   size_in_bytes_of_my_Ogg_file_image);

   printf("Valid Ogg data, if any, begins at %X\n",info.start_Ogg_data);

   while (AIL_enumerate_Ogg_pages(&info))
      {
      printf("Valid page found at byte offset %d from info.start_Ogg_data\n",info.byte_offset);
      }

See the OGG_INFO structure description for details about the information retrieved by this function.
