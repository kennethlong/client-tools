// Function prototype:

void AIL_set_sample_is_3D(HSAMPLE S,
S32 onoff);
Description

sets a sample from 2D to 3D or back again.

Parameters
S [in]

specifies the sample.
onoff [in]

specifies whether to turn on 3D spatialization, or off and return the sample to 2D control.
Return value
return [out]

Discussion

Miles HSAMPLEs are either 2D standard sound sources, or they are 3D sound sources and spatialized based on their distance from a listener (among other attributes). A sample will be placed in 3D mode once you call AIL_set_sample_3D_position on it, so this function is usually used to turn off 3D positioning (by passing 0 for the onoff parameter).
