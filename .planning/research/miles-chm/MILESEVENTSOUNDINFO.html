// Structure definition:
struct MILESEVENTSOUNDINFO
{
    U64 QueuedID;
    U64 InstanceID;
    U64 EventID;
    HSAMPLE Sample;
    HSTREAM Stream;
    void* UserBuffer;
    S32 UserBufferLen;
    S32 Status;
    U32 Flags;
    S32 UsedDelay;
    F32 UsedVolume;
    F32 UsedPitch;
    char const* UsedSound;
    S32 HasCompletionEvent;
};
Description

Sound instance data that is associated with each active sound instance.

Members
QueuedID [out]

A unique ID that identifies the queued event that started this sound. Returned from each call to AIL_enqueue_event.
InstanceID [out]

A unique ID that identified this specific sound instance (note that one QueuedID can trigger multiple InstanceIDs).
EventID [out]

A unique ID that identifies the actual event that started this sound. This is the same as QueuedID unless the sound was started by a completion event or a event exec step. In that case, the QueuedID represents the ID returned from AIL_enqueue_event, and EventID represents the completion event.
Sample [out]

The HSAMPLE for this playing sound.
Stream [out]

The HSTREAM for this playing sound (if it is being streamed, zero otherwise).
UserBuffer [out]

A pointer to the user data for this sound instance.
UserBufferLen [out]

The length in bytes of the user data (if known by Miles).
Status [out]

One of the MILESEVENTSOUNDSTATUS status values.
Flags [out]

One or more of the MILESEVENTSOUNDFLAG flags.
UsedDelay [out]

The value actually used as a result of the randomization of delay for this instance
UsedVolume [out]

The value actually used as a result of the randomization of pitch for this instance
UsedPitch [out]

The value actually used as a result of the randomization of volume for this instance
UsedSound [out]

The name of the sound used as a result of randomization. This pointer should NOT be deleted and is only valid for the until the next call in to Miles.
HasCompletionEvent [out]

Nonzero if the sound will fire an event upon completion.
Discussion

This structure is returned by the AIL_enumerate_sound_instances function. It returns information about an active sound instance.
