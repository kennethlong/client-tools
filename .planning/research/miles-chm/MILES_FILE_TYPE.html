// Type Definition:
typedef S32 MILES_FILE_TYPE;
Description

describes the type of a file.

Predefined values
AILFILETYPE_UNKNOWN 

File isn't one of the supported types. (Debugger value: 0).
AILFILETYPE_PCM_WAV 

Standard PCM wave file. (Debugger value: 1).
AILFILETYPE_ADPCM_WAV 

IMA ADPCM compressed wave file. (Debugger value: 2).
AILFILETYPE_OTHER_WAV 

Unsupported compressed wave file. (Debugger value: 3).
AILFILETYPE_OTHER_ASI_WAV 

Compressed wave file that is handled by an unspecified ASI decoder. (Debugger value: 14).
AILFILETYPE_VOC 

Creative VOC digital sound file. (Debugger value: 4).
AILFILETYPE_MIDI 

Standard MIDI file. (Debugger value: 5).
AILFILETYPE_XMIDI 

XMIDI file. (Debugger value: 6).
AILFILETYPE_XMIDI_DLS 

XMIDI file containing embedded, uncompressed DLS data. (Debugger value: 7).
AILFILETYPE_XMIDI_MLS 

XMIDI file containing embedded, compressed DLS data. (Debugger value: 8).
AILFILETYPE_DLS 

Uncompressed DLS file. (Debugger value: 9).
AILFILETYPE_MLS 

Compressed DLS file. (Debugger value: 10).
AILFILETYPE_MPEG_L1_AUDIO 

Compressed MPEG Layer 1 file (which MSS cannot play). (Debugger value: 11).
AILFILETYPE_MPEG_L2_AUDIO 

Compressed MPEG Layer 2 file (which MSS cannot play). (Debugger value: 12).
AILFILETYPE_MPEG_L3_AUDIO 

Compressed MPEG Layer 3 file. (Debugger value: 13).
AILFILETYPE_OGG_VORBIS 

Compressed Ogg Vorbis file. (Debugger value: 16).
AILFILETYPE_BINKA 

Compressed Bink Audio file. (Debugger value: 24).
AILFILETYPE_V12_VOICE 

Compressed Voxware V12 voice codec compressed file. (Debugger value: 17).
AILFILETYPE_V24_VOICE 

Compressed Voxware V24 voice codec compressed file. (Debugger value: 18).
AILFILETYPE_V29_VOICE 

Compressed Voxware V29 voice codec compressed file. (Debugger value: 19).
