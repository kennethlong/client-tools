U64 AIL_start_sound_instance(HMSOUNDBANK bank,
char const * sound,
U8 loop_count,
S32 should_stream,
char const * labels,
void* user_buffer,
S32 user_buffer_len,
S32 enqueue_flags);
Description

Allows the programmer to manually enqueue a start sound event into the event system.

Parameters
bank [in]

The bank containing the sound to start.
sound [in]

The name of the sound file to start, including bank name, e.g. "BankName/SoundName"
loop_count [in]

The loop count to assign to the sound. 0 for infinite, 1 for play once, or just the number of times to loop.
labels [in]

An optional comma-delimited list of labels to assign to the sound playback.
user_buffer [in]

See the user_buffer description in AIL_enqueue_event.
user_buffer_len [in]

See the user_buffer_len description in AIL_enqueue_event.
enqueue_flags [in]

See the enqueue_flags description in AIL_enqueue_event.
Return value
return [out]

Returns a non-zero EnqueueID on success.

Discussion

Enqueues an event to start the specified sound asset.

Usually the programmer should trigger an event that the sound designer has specifically create to start the appropriate sounds, but this function gives the programmer manual control, if necessary. This function is not captured by the auditioner.
