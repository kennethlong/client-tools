U32 AIL_inspect_BinkA(BINKA_INFO* state,
char const* file_image,
S32 file_size);
Description

obtains some basic information about the structure of an Bink Audio file, and prepares for optional frame enumeration.

Parameters
state [out]

points to an instance of a BINKA_INFO structure owned by the application.
file_image [in]

points to a BinkA file image owned by the application.
file_size [in]

The size, in bytes, of the BinkA file image.
Return value
return [out]

Returns 0 if the file was not a BinkA.

Discussion

This function finds basic information such as sample rate and overall length. In addition, the output state variable will be prepared for usage in AIL_enumerate_BinkA_frames.

See the BINKA_INFO structure description for details about the information retrieved by this function.
