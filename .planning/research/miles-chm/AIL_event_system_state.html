// Function prototype:

void AIL_event_system_state(HEVENTSYSTEM system,
MILESEVENTSTATE* state);
Description

Returns an information structure about the current state of the Miles Event System.

Parameters
system [in]

The system to retrieve information for, or zero for the default system.
state [in]

A pointer to a structure to receive the state information.
Discussion

This function is a debugging aid - it returns information for the event system.
