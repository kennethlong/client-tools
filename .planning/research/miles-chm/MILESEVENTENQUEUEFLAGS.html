// Type Definition:
typedef S32 MILESEVENTENQUEUEFLAGS;
Description

The available flags to pass in AIL_enqueue_event or AIL_enqueue_event_system.

Predefined values
MILESEVENT_ENQUEUE_BUFFER_PTR 

The user_buffer parameter passed in should not be duplicated, and instead should just tranparently pass the pointer on to the event, so that the MILESEVENTSOUNDINFO.UserBuffer during sound iteration is just the same pointer. user_buffer_len is ignored in this case. [Value: 0x1]
MILESEVENT_ENQUEUE_FREE_EVENT 

The ownership of the memory for the event is passed to the event system. If this is present, once the event completes AIL_mem_free_lock will be called on the raw pointer passed in to AIL_enqueue_event or AIL_enqueue_event_system. This is rarely used. [Value: 0x2]
MILESEVENT_ENQUEUE_BY_NAME 

The event passed in is actually a string. The event system will then look for this event in the loaded sound banks during queue processing. [Value: 0x4]
