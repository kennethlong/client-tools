// Function prototype:

void AIL_delay(S32 milliseconds);
Description

suspends foreground program execution.

Parameters
milliseconds [in]

specifies the number of milliseconds to wait.
Return value
return [out]

Discussion

This function may be used to provide a platform-independent delay, or other cases where the application needs to suspend processing for a brief amount of time.
