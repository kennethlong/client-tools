F32 AIL_3D_distance_factor(HDIGDRIVER driver);
Description

returns the driver's current 3D distance factor.

Parameters
driver [in]

indicates the driver to query.
Return value
return [out]

The driver's current global distance factor.

Discussion

The default distance factor is 1.0, which means your distances are specified in meters.
