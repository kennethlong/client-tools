// Type Definition:
typedef U32 MILES_FILE_SEEK_TYPE;
Description

describes the type of seek to perform.

Predefined values
AIL_FILE_SEEK_BEGIN 

Seek relative to the beginning of the file.
AIL_FILE_SEEK_CURRENT 

Seek relative to the current position of the file.
AIL_FILE_SEEK_END 

Seek relative to the end of the file.
