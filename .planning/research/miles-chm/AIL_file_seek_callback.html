// Function prototype:

typedef S32 AILCALLBACK AIL_file_seek_callback(UINTa file_handle,
S32 offset,
U32 type);
Description

specifies a user-replaceable callback handler to seek in a file.

Parameters
file_handle [in]

specifies the file handle to seek within.
offset [in]

specifies the offset to seek to.
type [in]

will be set to a MILES_FILE_SEEK_TYPE constant.
Return value
return [out]

This function should return the new absolute position of the file pointer (relative to the beginning of the file).

Discussion

Use these function definitions for your own file IO callbacks. Use AIL_set_file_callbacks to install them.
