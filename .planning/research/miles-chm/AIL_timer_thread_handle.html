// Function prototype:

S32 AIL_timer_thread_handle(void* value);
Description

retrieves the platform specific thread handle that Miles uses for the timer and mixing.

Parameters
value [out]

is a pointer to a variable that will receive the platform specific thread handle.
Return value
return [out]

Non-zero for success, 0 for failure.

Discussion

This function will return a platform specific handle for the Miles timer and mixer thread. You can use this function to move the thread to another core, or change it's priority (be careful if you change the thread priority).

On Windows and Xbox, this is a HANDLE. On PS3, this returns a sys_ppu_thread_t.
