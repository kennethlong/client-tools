// Function prototype:

typedef void AILEVENTERRORCB(S64 i_RelevantId,
char const* i_Resource);
Description

The function prototype to use for a callback that will be made when the event system encounters an unrecoverable error.

Parameters
i_RelevantId [in]

The ID of the asset that encountered the error, as best known. EventID or SoundID.
i_Resource [in]

A string representing the name of the resource the error is in regards to, or 0 if unknown.
Discussion

The error description can be retrieved via AIL_last_error.
