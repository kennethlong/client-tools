S32 AIL_audition_connect(char const* i_Address);
Description

Connect to a currently running Miles Sound Studio.

Parameters
i_Address [in]

The IP or host name of the computer running Miles Sound Studio. Use AIL_audition_local_host to connect to the same machine as the runtime.
Return value
return [out]

One of MILESAUDITIONCONNECTRESULT

Discussion

The is a synchronous connection attempt to Miles Sound Studio - it will not return until it is happy with the connection and the server, or a failure occurs.

This must be called before any AIL_add_soundbank calls.
