This filter is designed to provide side-chain style compression within the bus system.

Name for use with AIL_find_filter: "Compressor Filter"

Properties:

"Input": This is the signal strength to use for attuation calculations. The mixer automatically sets this, provided the compressor was installed via AIL_install_bus_compressor. Range: -32768 to 32767.

"Threshold": The decibel value, above which to start attentuating. Range: -96.0 to 0.0

"Ratio": The rate at which to attentuate the signal, once it exceeds the threshold. Range: 0 to 300.0

"Attack": The number of milliseconds the attenuation will take effect over. Range: 0 to 10000.0

"Release": The number of milliseconds the attenuation will cease over. Range: 0 to 10000.0
