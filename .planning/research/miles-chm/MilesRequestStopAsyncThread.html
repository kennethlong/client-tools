// Function prototype:

S32 MilesRequestStopAsyncThread(U32 spu_index);
Description

initiates the shutdown of a background SPU worker.

Parameters
spu_index [in]

Specifies the spu index to shutdown.
Return value
return [out]

Returns one on success, zero if there was an error.

Discussion

This function requests that Miles being closing one of the background SPU workers. This function doesn't wait for the function to exit - you must call MilesWaitStopAsyncThread to wait until the worker has completely closed.

You must call this function for any thread index that you called MilesStartAsyncThread on.
