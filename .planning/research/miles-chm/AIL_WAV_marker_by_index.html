// Function prototype:

S32 AIL_WAV_marker_by_index(void const * data,
S32 n,
char** name);
Description

returns a WAV marker offset based on an index ranging from 0 to the number of markers in the file.

Parameters
data [in]

points to a buffer containing the WAV file image.
n [in]

specifies the marker index.
name [out]

if not NULL, points to a variable of type char * which will receive a pointer to the marker name.
Return value
return [out]

The marker's byte offset in the WAV data chunk, or -1 if the call failed.

Discussion

The byte offset returned from this function may be passed directly to AIL_set_sample_position.
