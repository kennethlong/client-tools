The auditioning system allows the game to connect to Miles Studio and upload the state of the sound system each frame. We've designed this to be as easy as possible - but there are restrictions. In order to ensure that the auditioning system gets all the data it needs, you always need to enqueue events by name. This can be done with either the AIL_enqueue_event_by_name, or by passing MILESEVENT_ENQUEUE_BY_NAME in AIL_enqueue_event.

For an introduction to use it, see Integrating The Miles Event System.
