To distribute an application that uses Miles, you must include the Miles DLL (MSS32.DLL or MSS64.DLL) in the same directory as your application, and you must copy all of the other files in the \redist directory to a directory accessible by your application at run-time.

If your directory path will be a Unicode path, then see the FAQ on using Unicode.

Do not place the MSS32.DLL or MSS64.DLL file in the Windows or the Windows System directories! The Miles DLL must be placed in your application directory - it will refuse to be loaded from a system directory.
