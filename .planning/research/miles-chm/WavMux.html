The WavMux.EXE utility is a Windows console application that merges multiple stereo or mono files into one large multichannel wave file. This is useful as an input to BinkA_encode or to simply make uncompressed multichannel wave files from the command-line.

You can even use WavMux to create a PCM .WAV file that contains gaps in its channel mask. For instance, the command: 

      wavmux output.wav none none none none back_left.wav back_right.wav

...will leave gaps in the default channel mapping allowing you to automatically route each channel to the right speaker (you can also do this manually, at run-time).
