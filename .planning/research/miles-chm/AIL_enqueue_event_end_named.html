U64 AIL_enqueue_event_end_named(S32 token,
char const* event_name);
Description

Completes assembling the event and queues it to the command buffer to be run during next tick.

Parameters
token [in]

A token created with AIL_enqueue_event_start.
event_name [in]

The name of the event to run.
Return value
return [out]

A unique ID for the event that can be used to identify sounds started by this event, or for filtering future events to the sounds started by this event.

Discussion

This function takes all of the data accumulated via the various enqueue functions and assembles it in to the command buffer to be run during the next AIL_begin_event_queue_processing.

As with all of the enqueue functions it is completely thread-safe.

Upon completion of this function, the enqueue slot is release and available for another AIL_enqueue_event_start.
