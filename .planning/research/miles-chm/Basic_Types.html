Discussion

Miles strives to be as cross-platform as possible. To this end, Miles uses a set of basic types that are configured for each platform to ensure consistency in the API.

These basic types include simple integer types, float types, and more. In general, it is best if an application uses Miles' types internally in places where it interfaces to Miles, as this ensures that porting the application to other Miles platforms will be quick and easy.
