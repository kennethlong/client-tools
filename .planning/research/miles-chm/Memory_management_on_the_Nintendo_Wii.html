Before you can initialize Miles for Wii, you must first install memory callbacks uing the AIL_mem_use_malloc and AIL_mem_use_free functions. By default, there are no allocators installed on the Wii.

Miles does allocate from background threads on the Wii (when IO is performed in the background), so you need to initialize the Wii memory heaps as thread safe. Usually this is just a matter of using the MEM_HEAP_OPT_THREAD_SAFE flag to MEMCreateExpHeapEx.
