// Function prototype:

S32 AIL_enumerate_BinkA_frames(BINKA_INFO* state);
Description

obtains information about successive frames from a Bink Audio file.

Parameters
state [out]

points to an instance of an BINKA_INFO structure owned by the application and previously initialized by a call to AIL_inspect_BinkA.
Return value
return [out]

1 if the frame-data portion of the BINKA_INFO structure represents a valid frame; 0 if no (further) valid frames exist in the file image.

Discussion

This function returns information about all of the valid BinkA pages in the file image, beginning with the first, by filling in the appropriate BINKA_INFO structure fields.

You can inspect all of the frames in a BinkA file image with the following code: 

    BINKA_INFO info;
    if (AIL_inspect_BinkA(&info, (char const*)MyBinkAFileData, (S32)MyBinkAFileSize) == 0)
    {
        // Not a bink!
        return MSS_GENERAL_ERROR;
    }

    while (AIL_enumerate_BinkA_frames(&info))
    {
        printf("Page header at byte offset %d from MyBinkAFileData.\n", info.byte_offset);
    }

See the BINKA_INFO structure description for details about the information retrieved by this function.
