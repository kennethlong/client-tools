AILFREECB AIL_mem_use_free(AILFREECB freecb);
Description

replaces the default memory freeing routine.

Parameters
freecb [in]

specifies a memory free routine function of the AILFREECB type.
Return value
return [out]

The previous free routine's address.

Discussion

Use this function with AIL_mem_use_malloc.

Use this routine and AIL_mem_use_malloc to hook into your own memory management system.

AIL_mem_use_malloc and AIL_mem_use_free should be called prior to AIL_startup.
