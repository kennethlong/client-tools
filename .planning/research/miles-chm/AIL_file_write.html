// Function prototype:

S32 AIL_file_write(char const * filename,
void const * buf,
U32 len);
Description

creates a new data file and fills it with the specified data.

Parameters
filename [in]

specifies the data file that you wish to write to.
buf [in]

specifies the address to write from.
len [in]

specifies the number of bytes to write.
Return value
return [out]

Whether the function succeeded or not.

Discussion

Some platforms don't support writing data files (such as Wii), this function will fail safely, in this case.
