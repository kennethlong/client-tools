// Structure definition:
struct MILESBANKFUNCTIONS
{
    MilesBankFreeAll* FreeAll;
    MilesBankGetPreset* GetPreset;
    MilesBankGetEnvironment* GetEnvironment;
    MilesBankGetSound* GetSound;
    MilesBankGetEvent* GetEvent;
    MilesBankGetMarkerList* GetMarkerList;
    MilesBankGetLoadedCount* GetLoadedCount;
    MilesBankGetMemUsage* GetMemUsage;
    MilesBankGetLoadedName* GetLoadedName;
};
Description

specifies callbacks for each of the Miles event system.

Members
FreeAll [out]

Callback that tells you to free all user-side bank memory.
GetPreset [out]

Callback to retrieve a sound preset.
GetEnvironment [out]

Callback to retrieve an environment preset.
GetSound [out]

Callback to return the actual filename of a sound asset.
GetEvent [out]

Callback to retrieve a sound event.
GetMarkerList [out]

Callback to retrieve a sound marker list.
GetLoadedCount [out]

Callback to retrieve a count of loaded sound banks.
GetMemUsage [out]

Callback to retrieve the amount of memory in use.
GetLoadedName [out]

Callback to retrieve the filename for a sound asset index.
Discussion

This structure is used to provide overrides for all of the high-level loading functionality.
