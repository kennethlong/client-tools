void AIL_sample_51_volume_levels(HSAMPLE S,

Description

returns the current sample 5.1 volumes, from 0 to 1.0, on discrete multichannel platforms.

Parameters
S [in]

indicates the sample whose volume is to be retrieved.
front_left_level [out]

returns the front left speaker volume level from 0 (minimum) to 1.0 (maximum).
front_right_level [out]

returns the front right speaker volume level from 0 (minimum) to 1.0 (maximum).
back_left_level [out]

returns the back left speaker volume level from 0 (minimum) to 1.0 (maximum).
back_right_level [out]

returns the back right speaker volume level from 0 (minimum) to 1.0 (maximum).
center_level [out]

returns the center speaker volume level from 0 (minimum) to 1.0 (maximum).
sub_level [out]

returns the subwoofer speaker volume level from 0 (minimum) to 1.0 (maximum).
Return value
return [out]

Discussion

This function is available on all software-mixed platforms. Parameters that don't correspond to an available speaker in the HDIGDRIVER's current mode will be ignored.

Levels calculated by this function are multiplied with any values arising from 3D positioning or the use of the AIL_set_sample_channel_levels function. This function is intended primarily for use as a "shortcut" for dedicated 5.1-channel applications, notably on the Xbox 360 platform (which are always treated as 5.1-channel platforms by Miles).
