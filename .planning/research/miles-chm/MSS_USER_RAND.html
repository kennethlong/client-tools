// Function prototype:

typedef S32 AILCALLBACK MSS_USER_RAND();
Description

The function definition to use when defining your own random function.

Discussion

You can define a function with this prototype and pass it to AIL_register_random if you want to tie the Miles random calls in with your game's (for logging and such).
