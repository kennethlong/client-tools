// Function prototype:

SINTa AIL_set_preference(U32 name,
SINTa value);
Description

controls certain internal parameters used by the MSS system.

Parameters
name [in]

specifies one of the MILES_PREFERENCES to set.
value [in]

specifies the desired preference value.
Return value
return [out]

The preference value which was previously in effect.

Discussion

See the Preferences section for information on the meaning of various preference names and the effects their preference values have on the MSS system and drivers.

Example files that use this function: SSTEST.C
