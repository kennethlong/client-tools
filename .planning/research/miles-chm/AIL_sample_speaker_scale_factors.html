// Function prototype:

void AIL_sample_speaker_scale_factors(HSAMPLE S,
MSS_SPEAKER const * dest_speaker_indexes,
F32* levels,
S32 n_levels);
Description

retrieves the speaker scale factors for a sample.

Parameters
S [in]

indicates the HSAMPLE for which the speaker scale factors are to be queried.
dest_speaker_indexes [in]

specifies an array of one or more speakers.
levels [out]

specifies an array of where the current levels are stored for the speakers you pass in the the dest_speaker_indexes pointer.
n_levels [in]

specifies the number of entries in dest_speaker_indexes, and levels.
Return value
return [out]

Discussion

This function lets you retrieve the per-output speaker volume scale factors passed to AIL_set_sample_speaker_scale_factors by the application or established at sample-initialization time.

For a detailed understanding of per-channel level control, you should carefully review the How do I use multi-channel sound output in Miles? FAQ before attempting to use this function.
