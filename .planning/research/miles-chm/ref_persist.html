Overview

A persisted preset is a preset that is applied to a Label Query for all current and future sounds, until it is removed.

  1. Name of the persist, used to replace or remove it at a later time.
  2. Preset to apply. Name, clear, set. If not set, the named preset will be removed from all sounds it affects.
  3. The Label Query to affect.
  4. If checked, the preset will poll variables every frame and update accordingly.
