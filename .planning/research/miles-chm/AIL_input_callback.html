typedef void AILCALLBACK AIL_input_callback(void const * data,
S32 bytes,
UINTa user_data);
Description

specifies a user callback that is called as audio data is received by the input device.

Parameters
data [in]

points to a buffer containing the audio data.
bytes [in]

specifies the length in bytes of the audio buffer.
user_data [in]

specifies the user_data value from the input structure.
Return value
return [out]

Discussion

This function is called back from Miles as digital audio is received from the input device.
