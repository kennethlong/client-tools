void AIL_serve();
Description

provides periodic foreground service to the Miles Sound System to avoid "starvation" during periods when the application does not yield control frequently.

Return value
return [out]

Discussion

This function can be used to manually pump the Miles background processing - it's useful if the background thread is liable to become starved. It's normally only called in unusual situations where giving more CPU to the background thread isn't possible.
