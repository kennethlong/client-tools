void AIL_set_sample_playback_delay(HSAMPLE sample,
S32 playback_delay);
Description

sets an amount of time before the sound begins playing.

Parameters
sample [in]

specifies the digital sample that you wish to control.
playback_delay [in]

specifies the number of ms to wait before a sound begins playing.
Return value
return [out]

Discussion

This function allows you to set a sound delay that will start counting down once you call AIL_start_sample. This lets you start sounds in the future without having to monitor them.

Note that the status of a delayed sound is still SMP_PLAYING - it is playing, just the silent delay that you have set.
