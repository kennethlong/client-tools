Discussion

This filter has a unity-gain transfer function for all frequencies. It affects only the phase of the signal. An all-pass filter is not normally used as a standalone effect, since the human ear is insensitive to absolute phase. It is intended for use in specialized or experimental applications such as reverb signal chains.

The filter is implemented as a two-pole all-pass filter, constructed by taking twice the output of a bandpass filter and subtracting it from the original signal.

Name for use with AIL_find_filter: "AllPass Filter"

Properties:

"Allpass Center": Center Frequency. Range: 20.0 Hz to Nyquist (one-half the driver's output sample rate in Hz).

"Allpass Q": Determines the width of the phase-transition region (i.e., the band of frequencies where the phase shift goes from -90 degrees to -270 degrees). Range: 0.01 to 1.0, given as a linear percentage of the Allpass Center frequency parameter.

"Wet Level": Volume level of the processed data. Range: 0.0 (no effect) to 1.0 (full effect).

"Dry Level": Volume level of the original data. Range: 0.0 (no effect) to 1.0 (full volume).

When a mix value of 0.0 is specified, the filter will operate on an internal "fast path" designed to consume minimal CPU resources.
