// Function prototype:

S32 AIL_enqueue_event_filter(S32* token,
U64 apply_to_ID);
Description

Limits the effects of the event to sounds started by the given ID.

Parameters
token [in]

A token created with AIL_enqueue_event_start
apply_to_ID [in]

The ID to use for filtering. This can be either a sound or event ID. For an event, it will apply to all sounds started by the event, and any events queued by that event.
Return value
return [out]

0 if the enqueue buffer is full

Discussion

IDs are assigned to events and sounds - for events, it is returned via the AIL_enqueue_event_end_named function (or any other enqueue function). For sounds, you can access the assigned id during the enumeration process.
