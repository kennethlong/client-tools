Discussion

This filter is built right into the Miles mixer. See AIL_set_sample_low_pass_cut_off for details.

You can also install the low-pass filter as a normal pipeline-stage filter like any of the others, if you wish. This can be useful if you wish to add additional low-pass filtering to an occluded sample, for instance. In this case, the following properties are available:

Name for use with AIL_find_filter: "LowPass Filter"

"Lowpass Cutoff": Cutoff Frequency. Range: 20.0 Hz to Nyquist (one-half the driver's output sample rate in Hz).

"Wet Level": Volume level of the processed data. Range: 0.0 (no effect) to 1.0 (full effect).

"Dry Level": Volume level of the original data. Range: 0.0 (no effect) to 1.0 (full volume).

When a mix value of 0.0 is specified, the filter will operate on an internal "fast path" designed to consume minimal CPU resources.
