Q: Do you support MIDI controlled mixing faders?

A: Yes, the Miles Sound Studio supports the Behringer BCF2000 MIDI controller. It's much nicer to edit audio mixes this way (and the controller is only 179 bucks at Amazon). We can't recommend them enough. They are motorized faders, so all of your settings are restored each time you click on a preset.

To use the BCF2000, you need to plug it into a USB port and install the drivers that it ships with. Then, you need to start it in Logic Control mode - to do this, hold down the button in the top row, third from the left while flipping the power switch in the back. If you did it correctly, then the LEDs in the upper right will read "LC".

In Miles Studio 9, the mixer sliders are automatically bound to the corresponding Slider, from left to right. In order to use the mixer sliders, you need to create a slider (as described in the Dynamic Presets user guide). The mixer slider will automatically work at that point, and adjust the bound variable.
