// Function prototype:

void AIL_schedule_start_sample(HSAMPLE S,
U64 mix_time_to_start);
Description

Marks the specified sample to begin at the exact time specified.

Parameters
S [in]

The sample to start
mix_time_to_start [in]

The time to start the sample, in samples.
Discussion

Once set, the sample will have AIL_start_sample called automatically when the mixer reaches the specified time. The sample's delay will be automatically adjusted such that the sample starts mid-block. 

        // Get the current time.
        U64 mix_time = AIL_digital_mixed_samples(dig);

        // Schedule to start 1 second out
        mix_time += AIL_digital_samples_per_second(dig);
        AIL_schedule_start_sample(S, mix_time );
