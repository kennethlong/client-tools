void AIL_inspect_Ogg(OGG_INFO* inspection_state,
U8* Ogg_file_image,
S32 Ogg_image_size);
Description

prepares for enumeration of the page boundaries in an Ogg container file.

Parameters
inspection_state [out]

points to an instance of an OGG_INFO structure owned by the application.
Ogg_file_image [in]

points to an Ogg file image owned by the application.
Ogg_image_size [in]

The size, in bytes, of the Ogg file image.
Return value
return [out]

Discussion

This function prepares an OGG_INFO structure instance for use with AIL_enumerate_Ogg_pages.

See the OGG_INFO structure description for details about the information retrieved by this function.
