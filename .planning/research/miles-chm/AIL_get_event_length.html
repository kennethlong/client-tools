S32 AIL_get_event_length(char const* i_EventName);
Description

Returns the length of the first sound referenced in the named event, in milliseconds.

Parameters
i_EventName [in]

The name of an event that starts a sound.
Return value
return [out]

The length in milliseconds, or 0 if there is an error, or the event has no sound references, or the sound was not found.

Discussion

This looks up the given event and searches for the first Start Sound event step, then uses the first sound name in its list to look up the length. As such, if the start sound step has multiple sounds, the rest will be ignored.
