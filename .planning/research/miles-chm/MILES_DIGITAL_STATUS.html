// Type Definition:
typedef U32 MILES_DIGITAL_STATUS;
Description

describes the current state of the digital audio.

Predefined values
SMP_DONE 

Sample has finished playing, or is a sample and has never been started. Streams that haven't been started return SMP_STOPPED. (Debugger value: 2).
SMP_PLAYING 

Sample is currently playing (active). (Debugger value: 4).
SMP_STOPPED 

Sample has been stopped/paused. (Debugger value: 8).
SMP_FREE 

Sample has been freed back to the system. (Debugger value: 1).
