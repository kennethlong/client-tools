Q: How can I tell what version of Miles that I have?

A: It depends on when and what version you'd like to check.

At runtime, you can call the AIL_MSS_version function to get the library version.

You can get the header file version by looking at the top of the MSS.H file.

You can check the version of any of the Windows DLLs (including ASIs and FLTs) by right-clicking on the file in Explorer, choosing "Properties" and then clicking on the Version tab.

You can check the version of the Mac shared libraries (including ASIs and FLTs) by using the Finder menu option "Get Info...".
