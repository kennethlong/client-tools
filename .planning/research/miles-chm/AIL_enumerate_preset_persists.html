S32 AIL_enumerate_preset_persists(HEVENTSYSTEM system,
HMSSENUM* next,
char const ** name);
Description

Enumerates the current persisted presets that active in the system.

Parameters
system [in]

The system to enumerate the persists for, or 0 to use the default system.
next [in]

Enumeration token - initialize to MSS_FIRST before the first call.
name [out]

Pointer to a char* that receives the name of the persist. NOTE that this pointer can change frame to frame and should be immediately copied to a client-allocated buffer if persistence is desired.
Return value
return [out]

Returns 0 when enumeration is complete.

Discussion

This function lets you enumerate all the persisting presets that are currently active in the system. It is mostly a debugging aid.
