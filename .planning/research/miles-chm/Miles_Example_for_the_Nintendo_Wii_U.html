The MSS SDK supplies MilesWiiU.CPP, an example program for the Wii-U. This file will compile without modification in any Wii-U project structure. You will need to add references to any necessary MSS libraries: MSSWiiU.A, FLTWiiU.A, MP3WiiU.A, BinkAWiiU.a, OGGWiiU.A. (Only the MSSWiiU.A library is mandatory.)

In addition, MilesWiiUDRC.cpp details various ways of getting audio to play on the DRC unit.

Lastly, EventWiiU.cpp details how to connect to Miles Studio from the Wii-U.
