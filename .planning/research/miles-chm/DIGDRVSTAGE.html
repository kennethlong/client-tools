// Type Definition:
typedef U32 DIGDRVSTAGE;
Description

describes the stage in the driver pipeline to install the filter on.

Predefined values
DP_FLUSH 

Sets the provider to be called when the mixing buffer is to be cleared to silence (before the DP_MERGE stage is called on each HSAMPLE).
DP_DEFAULT_FILTER 

Sets the default filter provider for this HDIGDRIVER. Once set, each call to AIL_init_sample will set the sample's first filter provider (DP_FILTER_0) to this value.
DP_DEFAULT_MERGE 

Sets the merge provider for this HDIGDRIVER. The merge provider is called to mix each HSAMPLE's current address into the internal MSS mixing build buffer.
DP_COPY 

Sets the provider to be called when all mixing is completed and the internal MSS mixing build buffer needs to be scaled down from 32-bits into the final output buffer.
DP_MC_COPY 

Sets the provider to be called when all mixing is completed and the internal MSS mixing build buffers need to be scaled down from 32-bits into the final multi-channel output buffer.
