// Function prototype:

U32 AIL_event_system_command_queue_remaining();
Description

Returns the number of bytes remaining in the command buffer.

Discussion

This can be invalid for a number of reasons - first, if the command buffer will need to wrap for the next queue, the effective bytes remaining will be lower. Second, if an enqueue occurs on another thread in the interim, the value will be outdated.
