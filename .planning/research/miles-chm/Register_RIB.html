// Function prototype:

void Register_RIB(void* static_lib_name);
Description

registers a RIB interface on static library platforms such as Wii, PS3, PSP, or Xbox 360.

Parameters
static_lib_name [in]

Name of the static RIB library to load.
Return value
return [out]

Discussion

On the static library versions of Miles (Xbox 360, Wii, PS3, PSP), you must manually load the ASI decompressors that you want to use.

This function (macro actually) takes a single parameter - the static library to load. Currently, the available static libraries include MP3Dec, BinkADec, OggDec, Voice, SRS, DolbySurround, and DSP. To load any or all of these libraries, you use this function before AIL_startup. For example: 

Register_RIB(BinkADec);
Register_RIB(MP3Dec);
Register_RIB(DSP);
Register_RIB(Voice);
Register_RIB(XMADec);
Register_RIB(OggDec);

AIL_startup();
