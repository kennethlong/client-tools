Overview

Defaults are how you manage per platform compression settings without having to create a child Sound asset for each target platform. When you compile a bank for a different platform (or you select a different audition platform), Miles Studio will recognize that the compression settings have changed and invalidate the sound asset, causing recompression with the settings for the desired platform. Note that in order for this to occur, the sound asset's compression has to be set to "Use Defaults".

To set up Defaults, open the Bank containing your Settings, or create a new bank. Select the Defaults item under Settings, and click Create Defaults.

Initially, global compression settings are displayed. These will be applied to all sounds with "Use Defaults" selected, no matter what platform is being compiled. These compression settings are the same as the ones described in the Sound Assets documentation.

To add an override for a specific platform, select the desired platform from the "Select Platform To Add" drop down, and set the compression settings as desired.
