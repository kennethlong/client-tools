// Function prototype:

S32 AIL_WAV_info(void const * data,
AILSOUNDINFO* info);
Description

retrieves the audio format information for the specified file image.

Parameters
data [in]

points to a buffer containing the wave file image.
info [out]

is a pointer to an AILSOUNDINFO structure that will return the wave information.
Return value
return [out]

1, if the call was successful, or 0 if the call failed.

Discussion

This function can be used to load the info structure that is needed in the AIL_decompress_ADPCM function.

The channel_mask member corresponds to the dwChannelMask member of the WAVEFORMATEXTENSIBLE header defined by Microsoft. See the How do I use multi-channel sound output in Miles? FAQ for more on multichannel content authoring and channel masks.
