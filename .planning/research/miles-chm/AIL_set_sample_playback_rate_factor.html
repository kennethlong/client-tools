void AIL_set_sample_playback_rate_factor(HSAMPLE S,
F32 playback_rate);
Description

sets a sample's relative playback rate as a fraction of its absolute sample rate.

Parameters
S [in]

indicates the sample whose playback rate is to be set.
playback_rate [in]

specifies the sample's playback rate as a fraction of its absolute sample rate as given by AIL_sample_playback_rate.
Return value
return [out]

Discussion

Most applications that alter a sample's playback rate will find it more convenient to use AIL_set_sample_playback_rate_factor than AIL_set_sample_playback_rate. This function avoids the need to keep track of the sample's original absolute playback rate.

The default relative playback rate fraction is 1.0, which causes the file to play at the rate given by AIL_sample_playback_rate.
