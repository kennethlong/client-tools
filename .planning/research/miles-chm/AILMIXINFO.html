struct AILMIXINFO
{

Description

Members
Info [in]

Contains the details on each audio buffer that you'd like to mix together.
Discussion

The only part of the AILMIXINFO structure that you need to set is the Info structure inside the AILMIXINFO structure. The Info structure is of type AILSOUNDINFO, so you can use AIL_WAV_info to set the appropriate fields. See the example below for details.
