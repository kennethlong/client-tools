// Structure definition:
struct EAX_FSHIFTER
{
    S32 Slot;
    S32 Volume;
    F32 Frequency;
    U32 LeftDirection;
    U32 RightDirection;
};
Description

used to control the frequency shifter filter in the EAX 4 3D provider

Members
Slot [in]

specifies the slot to query or set. The frequency shifter can use slot 2 and/or 3.
Volume [in]

specifies the volume of this slot in standard EAX attenuation units (-10000 to 0 fixed point millibels, default 0.0).
Frequency [in]

specifies the frequency in hertz (0.0 to 24000.0, default 0.0).
LeftDirection [in]

specifies the left speaker direction - 0 = down, 1 = up, 2 = off.
RightDirection [in]

specifies the right speaker direction - 0 = down, 1 = up, 2 = off.
Discussion

See the EAX 4 documentation for technical details on this filter.
