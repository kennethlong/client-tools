This filter will record the final mixed output of the Miles digital audio system to a wave file, as long as a per-voice output filter like DirectSound3D/EAX is not in use. This is great for recording demos or tracking down bugs. Just set the filename and then apply the filter to the digital driver to begin recording.

Name for use with AIL_find_filter: "Capture Filter"

Properties:

"Filename": Name of the output file (pointer to zero-terminated string).

"Enabled": Recording enabled/disabled state. Range: Zero to inhibit recording; nonzero to enable recording.

See MULTIFLT.CPP for an example of how to use this filter.
