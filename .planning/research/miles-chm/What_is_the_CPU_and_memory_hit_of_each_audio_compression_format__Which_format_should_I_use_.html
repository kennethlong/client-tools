Q: What is the CPU and memory hit of each audio compression format? Which format should I use?

A: The CPU and memory hit of each format varies per-platform, as well as per sound format (44 Khz takes more than 22 Khz, for example). To make some rough guidelines, we've made two tables - one for sound effects (mono at 22 Khz), and one for music (stereo at 44 Khz).

As for which format you should use, we recommend ADPCM or Bink-Audio for sound effects, and Bink Audio or MP3 for music. You should rarely use uncompressed samples - the memory can be put to better use.

Uncompressed and ADPCM use about the same amount of CPU - ADPCM takes cycles to decode, but saves the memory hit. Both are fast enough, that we don't bother listing them in the tables below - it's just too low to care about (on a PC it takes 256 samples to hit 1% CPU, on an Xbox it takes 64 for 1%).
