Defines a ramp for use in the game. Ramps are means of smoothly interpolating aspects of a sound over time.

  1. The Label Query to use for determining sounds to affect.
  2. The value the ramp will have when the game loads. If a variable is assigned, the variable must be set before the first tick of the event system.
  3. The aspect of the sound to affect.

    See also Ducking and Ramps
