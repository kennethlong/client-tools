// Structure definition:
struct MILESEVENTSTATE
{
    S32 CommandBufferSize;
    S32 HeapSize;
    S32 HeapRemaining;
    S32 LoadedSoundCount;
    S32 PlayingSoundCount;
    S32 LoadedBankCount;
    S32 PersistCount;
    S32 SoundBankManagementMemory;
    S32 SoundDataMemory;
};
Description

returns the current state of the Miles Event System.

Members
CommandBufferSize [out]

The size of the command buffer in bytes. See also the AIL_startup_event_system.
HeapSize [out]

The total size of memory used by the event system for management structures, and is allocated during startup. This does not include loaded file sizes.
HeapRemaining [out]

The number of bytes in HeapSize that is remaining.
LoadedSoundCount [out]

The number of sounds loaded and ready to play via cache event steps.
PlayingSoundCount [out]

The number of sounds currently playing via start sound event steps.
LoadedBankCount [out]

The number of sound banks loaded in the system via cache event steps, or AIL_add_soundbank.
PersistCount [out]

The number of presets persisted via the persist event step.
SoundBankManagementMemory [out]

The number of bytes used for the management of the loaded sound banks.
SoundDataMemory [out]

The number of bytes used in file sizes - remember this is not included in HeapSize. Streaming overhead is not included in this number, only fully loaded sounds.
Discussion

This structure returns debugging info about the event system. It is used with AIL_event_system_state.
