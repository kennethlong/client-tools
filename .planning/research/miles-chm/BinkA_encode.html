The BinkA_encode.EXE utility is a Windows console application that compresses uncompressed PCM .WAV files into compressed Bink Audio format files.

Bink Audio is nice because it is super-fast to decode (on some platforms, twice as fast as MP3, and three times as fast as Ogg), has competitive compression ratios (8-12 to 1), is natively VBR (nice for dialog), uses little memory, supports multi-channel sounds natively, and doesn't have the IP issues that MP3 or Ogg entangle. It's a great format for games, and we expect it will be the default for more Miles users.

The encoder is really easy to use, it takes an input .WAV file (mono, stereo, or multichannel) and an output Bink Audio filename, and an optional compression level.

If you have a bunch of separate wave files that you want to turn into one Bink Audio file, then you can use our WavMux command line utility to merge them all.

You can run the binka_encode from the command line to see the command format.
