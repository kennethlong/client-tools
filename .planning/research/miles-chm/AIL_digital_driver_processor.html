HPROVIDER AIL_digital_driver_processor(HDIGDRIVER dig,
DIGDRVSTAGE pipeline_stage);
Description

returns the current default filter or mixer stage provider for a specific driver.

Parameters
dig [in]

specifies the HDIGDRIVER to query.
pipeline_stage [in]

specifies the stage to query. Use one of the SAMPLESTAGE constants.
Return value
return [out]

The currently-installed provider.
