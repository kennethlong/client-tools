// Function prototype:

void AIL_close_filter(HDRIVERSTATE filter);
Description

closes a previously opened filter.

Parameters
filter [in]

specifies the filter to close.
Return value
return [out]

Discussion

Open filters are cleaned-up automatically at driver shutdown time, so it is only necessary to explicitly close a filter if you wish to reclaim resources consumed by a filter than you no longer intend to use.

Example files that use this function: filter.cpp, multiflt.cpp
