Label Queries are how you select a group of sounds from the group of currently playing samples. When you edit a query, you add a label to the query, and you can mark it as "Must Match", "Can't Match", or "Can Match". By default, it's "Can Match". This means that if a sound has this label, it will match. "Must Match" means that the sound must have the label in its list. So if a sound has the labels "one", and the query is "Can Match one, Must Match two", the sound will fail to match.

The last is "Can't Match", which means that if the label is on a sound, it is automatically excluded.

An empty label query matches all sounds.
