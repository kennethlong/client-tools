This filter allows the you to smoothly ramp up a sound signal from one volume level to another. The volume is slowly adjusted on a sample by sample basis for completely smooth ramping. To use this filter, you first set the ramp time, and then you set the ramp level. The volume ramping will begin as soon as you set the ramp level. To set the initial ramp level to ramp from, you set the ramp time to zero first. So, to smoothly ramp the volume up from 0.0 to 1.0, use something like this: 

// First, set the ramp time to zero, to init the ramp level
F32 ramp_time = 0.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp Time", -1, 0, &ramp_time, 0 );
//
// we want start at level zero
F32 ramp_level = 0.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp To", -1, 0, &ramp_level, 0 );
//
//
//ok now, set the ramp time to 1000 ms (fade in over 1 second)
ramp_time = 1000.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp Time", -1, 0, &ramp_time, 0 );
//
// ramp up to 1.0 (full volume)
F32 ramp_level = 1.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp To", -1, 0, &ramp_level, 0 );

Similarly, to fade out over one second, do something like this: 

// First, set the ramp time to zero, to init the ramp level
F32 ramp_time = 0.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp Time", -1, 0, &ramp_time, 0 );
//
// we want start at full volume
F32 ramp_level = 1.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp To", -1, 0, &ramp_level, 0 );
//
//
//ok now, set the ramp time to 1000 ms (fade out over 1 second)
ramp_time = 1000.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp Time", -1, 0, &ramp_time, 0 );
//
// ramp down to 0.0 (silence)
F32 ramp_level = 0.0f;
AIL_sample_stage_property( S, SP_FILTER_0, "Ramp To", -1, 0, &ramp_level, 0 );

Note that the filter remembers the last ramp level, so you only have to set the starting ramp level if you don't already know what it is. So, if you ramp from 0.0 to 1.0, and then later want to wrap back down to 0.0, then you don't need to set the initial ramp level to 1.0 for the fade out.

Name for use with AIL_find_filter: "Volume Ramp Filter"

Properties:

"Ramp Time": Time to apply the new ramp level over. Range: 0.0 to milliseconds. So, to fade in over 1 second, use 1000.

"Ramp To": The new volume level to smoothly ramp to. Range: 0.0 (silence) to 1.0 (full volume).

properties:

"Ramp At": Returns the current level of the volume ramping. This property will change over the ramp time until it matchs the "Ramp To" value. You can poll this property to determine when a volume ramp is complete. Range: 0.0 (silence) to 1.0 (full volume).
