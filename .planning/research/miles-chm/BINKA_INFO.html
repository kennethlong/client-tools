// Structure definition:
struct BINKA_INFO
{
    char const * file_image;
    S32 image_size;
    S32 total_samples;
    S32 channel_count;
    S32 sample_rate;
    S32 samples_per_frame;
    char const * current_frame;
    S32 byte_offset;
};
Description

Members
file_image [in]

Original pointer passed to AIL_inspect_BinkA.
image_size [in]

Original file_size passed to AIL_inspect_BinkA.
total_samples [in]

Total samples in the file. Divide by sample_rate to get length in seconds.
channel_count [in]

Total channel count in the file.
sample_rate [in]

The sample rate for the file.
samples_per_frame [in]

The samples contained in one frame enumerated via AIL_enumerate_BinkA_frames.
current_frame [in]

Internal pointer used for enumeration.
byte_offset [in]

Offset of the frame from file_image, suitable for a loop point.
Discussion

All fields except byte_offset are valid as soon as AIL_inspect_BinkA returns. byte_offset is valid once an enumeration has taken place.
