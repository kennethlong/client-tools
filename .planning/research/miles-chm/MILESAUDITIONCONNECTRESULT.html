typedef S32 MILESAUDITIONCONNECTRESULT;
Description

Return values for AIL_audition_connect.

Predefined values
MILES_CONNECTED 

The Auditioner connected and successfully executed the handshake. [Value: 0]
MILES_CONNECT_FAILED 

The Auditioner couldn't connect - either the IP wasn't valid, or Miles Sound Studio wasn't accepting connections. [Value: 1]
MILES_HOST_NOT_FOUND 

The given host name could not be resolved to an IP. [Value: 2]
MILES_SERVER_ERROR 

We connected, but the server was either another app on the same port, or the server version was incorrect. [Value: 3]
