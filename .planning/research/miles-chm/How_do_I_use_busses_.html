Q: How do I use busses?

A: Allocate a bus, then assign samples to it.

  1. Call AIL_allocate_bus to create an additional bus.
  2. Call AIL_set_sample_bus to assign a sample to a bus. By default, samples are on the Main Bus (index 0).
  3. Call AIL_bus_sample_handle to retrieve the sample associated with a bus, if desired.

Sounds are then mixed to an intermediate bus prior to output to the device. This bus is just another HSAMPLE, and can be adjusted in the same way as in any normal HSAMPLE, with a few restrictions. For example, you can set a filter on the bus sample, but you can not adjust the playback rate.

To install a side chain compressor filter, use AIL_install_bus_compressor. This is useful for automatic ducking.
